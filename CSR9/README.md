# fullstack_developer_capstone

Welcome to the Best Cars Dealership project! This application provides a platform to view dealerships, read and add reviews, and manage your account.

## Features
- View all dealerships
- Filter dealerships by state
- Read reviews for a specific dealer
- Add a review (requires login)
- User authentication (Login/Register)

## Setup
1. Clone the repository
2. Install dependencies
3. Run the Django server
